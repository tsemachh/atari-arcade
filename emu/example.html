AltirraSDL — embed example

A reference page for wiring the WASM Atari emulator into your own site.

Before this page does anything: place an Atari title at library/yourgame.atr next to this file. The iframe below points at library/yourgame.atr by default — change the Game file input below to use a different name, or edit the src attribute directly in the HTML source.

Copy the iframe tag above into your own page once you have the URL you want. The same URL also works as a direct link if you'd rather visitors land on the emulator full-frame.

What you can change

  • Multiple disks — pass a comma-separated list: lib=disk1.atr,disk2.atr.
  • Custom kernel ROM — drop the file into firmware/, then add &firmware=myos.rom to the URL. The emulator identifies it by hash.
  • Force a specific kernel — add &kernel=lle (built-in, no firmware needed) or &kernel=osb if you shipped an OS-B ROM.
  • Memory size — add &memsize=128K for a 130XE-only title.
  • Title — add &title=My+Game if you ever expose chrome.
  • Pixel-perfect text — add &crt=0&filter=point&artifact=none&stretch=integral and use aspect-ratio: 7 / 5 for PAL or 3 / 2 for NTSC. See EMBED.md for why 4 / 3 isn't quite right.
  • Lazy load (click-to-play) — add &autoplay=0 to skip the wasm download until the visitor clicks the canvas. Saves ~5 MB per visitor who never engages.
  • Open in the full emulator UI — add &ui=desktop (and omit embed=1) for a "try this in the full emulator" link.
  • Period-accurate look — add &experience=authentic&addons=off for a stock 800XL with PAL timing by default, drive sounds, and accurate disk timing. Add &ntsc=1 for North American titles. See the "First-run defaults" section in EMBED.md for the full matrix.
